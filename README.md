# CoverCharm

### Development Goals
- This is a web application with a full-stack experience
    - Front-end: [Next.js](https://angular.dev/)
    - Middle-level: [Django](https://www.djangoproject.com/)
    - Backend: MySQL

### How To Run Locally
1. Install Node.js
```
npm -v
```
2. Start the frontend
```
npm install
cd client
npm run dev
```
